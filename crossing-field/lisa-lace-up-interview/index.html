2011年4月，LiSA 攜迷你專輯《Letters to U》踏上獨立藝人之路。迎來出道15週年的她，完成了第7張原創全長專輯《LACE UP》。即便已因動畫《鬼滅之刃》系列主題曲〈紅蓮華〉、〈炎〉而廣為人知，LiSA 對音樂的貪婪姿態從未停歇。新作中透過與ケンモチヒデフミ、GLIM SPANKY、水野良樹（いきものがかり）等藝術家的全新相遇，進一步拓展了音樂的寬度。

始終將聽眾，也就是「LiSAっ子」們，放在心上，帶著他們一同前行、共享歡樂，LiSA 就這樣走過了15年。這個值得紀念的週年，想必將帶來比以往更多的驚喜與歡樂。這次是音樂ナタリー第25次專訪，以滿載期待的專輯《LACE UP》以及將在東京日本武道館揭幕的15週年巡演「LiVE is Smile Always～15～（iCHiNiTSUiTE GO）」為主軸，深入探尋 LiSA 作為 LiSA 而活著的那股充沛能量之源。

以15週年為目標，固守 LiSA 核心的一年及其踏實感

——2025年有5月的日本武道館公演「RiP SERViCE」，以及從9月延續至今年1月的廳堂巡演「PATCH WALK」。我覺得身為出道14週年的去年，整體動向像是已在朝著15週年邁進，帶著對今年展開的預期意識。實際上是這樣嗎？

進入14週年之前，2024年底結束了巡迴演唱會「COCKTAiL PARTY」之後……怎麼說呢，我覺得自己用一種過猛的速度衝過了那個終於變得乾淨的世界。

——那正是歷經新冠疫情之後，終於能夠重回從前那種現場演出的時期，對吧。

對。因為帶著一種「必須把這幾年沒辦法做的事全部補回來」的心情衝過去，結果有點累了。意識到這件事大概是2025年初。我想，這樣下去15週年會一眨眼就過去，所以必須好好整頓一下再出發，得先確認自己腳下踩的地方。稍微請了點假，把時間用來沉澱思考如何迎戰15週年。15週年一到，不管願不願意都會來到必須回顧歷史的時刻，所以我想趁那之前先把自己在現場演出中所建立的核心部分鞏固好。抱著這樣的心情而做的，就是「RiP SERViCE」和「PATCH WALK」這兩場演出。

——與其說是為15週年蓄勢，更像是踏踏實實地確認腳下。所有公演結束之後，你覺得成功了嗎？

成功了。

「RiP SERViCE」的時候就有很強烈的踏實感。在大型場館演出，需要增添讓更多人感受到的演出效果、安排縝密的流程。大場地裡，舞台機關的運作、層次差異的製造，加入無法單靠音樂引導觀眾的演出也很重要。但「RiP SERViCE」借助了武道館這個空間的優勢，讓我有了自信，我們只靠自己就能盡力傳達了。

——「RiP SERViCE」中雖然演唱了「明け星」和「炎」，但「紅蓮華」或「Rising Hope」等常見的單曲主打並未出現，整體是 B 面曲偏多的構成。

在武道館兩天都搞 B 面，我也想過這樣真的好嗎（笑）。

——最近才對 LiSA 產生興趣而來的人，可能會嚇一跳吧。

對對。沒有一首認識的歌，大概就是這種感覺（笑）。「PATCH WALK」巡演也幾乎是滿滿的 B 面。我相信自己在現場演出中所培育的那些歌的力量，相信靠歌的力量就能帶動觀眾，就帶著這份信念走完了全18場，確實感受到了踏實的回應。

——2025年的收尾，還有時隔4年重登「NHK紅白歌合戰」。在那場記者會上，LiSA 以「繋（連結）」作為「今年的漢字」，並表示：「今年以拼布縫製為主題，把各種回憶與足跡串聯起來作為活動方向，而紅白歌合戰的主題正是『つなぐ、つながる、大みそか。』，讓我感受到一種命運的相繫。」真是漂亮的著陸。

是啊。2025年初所感受到的不安消失了，心想「沒問題了」。

自己也能說喜歡 LiSA 的自己

——就這樣確認好腳下，同時著手製作新專輯《LACE UP》。「LACE UP」有「編織而成」的意涵嗎？

是的。「LACE UP」是編織的意思——我始終穿著 Doc Martens 綁好鞋帶，成為 LiSA 走向舞台。

我其實從「RiP SERViCE」就開始自顧自地把故事串聯起來了。裁切（Rip）、拼布縫製、然後全部結繫起來，就像製作一件衣服的過程。我希望這一年能成為通往專輯的旅程，帶著這樣的心情在製作。

——從出道10週年迷你專輯《LADYBUG》前後開始，音樂性的寬度就顯著擴展，而這張專輯的印象是又進一步延伸了。不只是音樂類型的廣度，就連在應是拿手的搖滾領域，也在推進那些過去尚未嘗試的搖滾。

確實如此。光是把已發行的單曲排在一起，就已經往很多不同方向延伸開來，所以我心想：要把這些收攏成一個整體可真不容易。要用「LiSA」這條軸心把已經展開的東西結繫在一起，就必須去填補其中的空隙。在一開始說到的、去年初請了短暫假期的那段時間誕生的，正是「Patch Walk」這首歌。從那裡浮現了「LACE UP」這個標題，整件事也有了一條脈絡。從那個時刻起，我覺得這張專輯沒問題了。

——「Patch Walk」收錄在專輯的最後。我認為它在某種意義上是這張作品的主題曲，第一次聽到時的印象是：這首歌的規模感，是透過與「Another Great Day!!」合作的 B'z Tak Matsumoto 所獲得的東西。

是的是的，我懂你的感覺。

——同時，「『好き』を縫い合わせて 仕立てた私（將喜歡的事縫合，剪裁出的我）」這句歌詞，凝縮著15年的真實感受。而「歩き続けて また会いましょう（繼續走下去，再相見吧）」正是此刻 LiSA 本人的宣言。

請了那段假期的時候，我讓自己重新面對 LiSA 的音樂，思考今後應該如何前行。那自然也是必須去解開自己過往的時間。在那之後之所以能覺得「沒問題了」，是因為只是不偷懶地走著，就已來到如此了不起的地方，所選擇的道路也沒有錯，而且我自己也能說喜歡 LiSA 了。把這種心情寫下來就成了「Patch Walk」。從那裡，專輯的主題也浮現出來……我的專輯標題都是「L」開頭的（笑），思考了各種 L 開頭的詞彙之後，「LACE UP」浮現了，就覺得這樣沒問題了。

——真的是這張專輯的核心之歌。「Patch Walk」的作曲是與堀江晶太聯名，編曲也是他。

晶太一直都在 LiSA 故事的身旁看著，他是唯一能夠近距離與我對話、討論 LiSA 接下來該做什麼的音樂人。請假的時候我請他一起去了美國。帶著木吉他，「一起來看這片風景吧」（笑）。在廣闊的大地和清爽的空氣中，一邊交談著我想做的事，一邊完成了這首歌。

——這首歌在「PATCH WALK」巡演中搶先披露，提早讓大家聽到了。

大家在演唱會現場第一次聽到這首歌。我很想好好傳達歌詞，所以在演出結尾片尾所播放的影像中加入了歌詞一起讓大家聽。我希望那些15年來一直相信 LiSA、跟隨而來的大家，能夠感受到「LiSA 好像不會消失」「可以一起帶著期待迎向15週年」。在演出後段演唱，片尾還讓大家再聽一遍，我想這樣應該傳達到了。

將15年的璀璨裝上裝飾卡車出發

——專輯開頭的「OPENiNG -LACE UP-」接「DECOTORA15」，這兩首也是新錄製的新曲。純器樂的「OPENiNG -LACE UP-」從靜謐的開場展開為華麗的銅管搖滾，銜接帶有流行龐克氣勢的「DECOTORA15」。這個開場設定了什麼樣的意義？

「LACE UP」是編織的意思，而我上台時始終穿著 Doc Martens 的編織靴。我一直都是穿著 Doc Martens 綁好鞋帶，成為 LiSA 走向舞台。這是準備階段振奮精神、作為 LiSA 不可或缺的儀式。「OPENiNG -LACE UP-」裡，「Believe in myself」「best day, best way」「ジェットロケット」「ROCK-mode」這些在演出中串聯過的歌曲的記憶片段，由帶有慶典感的喇叭聲音串起。從那裡，將大家的思念裝飾一番，搭上裝飾卡車（デコトラ）閃亮出發，就是這樣的意象。

——把15年的思念都裝上去，就成了裝飾卡車。

對（笑）。裝飾過的貨車，歌曲收錄了15首，所以就是帶著這15首歌、以15週年為起點出發的意思。

——「DECOTORA 15」我聽著就覺得是首好歌，「作曲家作了不錯的工作呢」，一看作曲欄才發現是 LiSA 本人。

謝謝（笑）。龐克、流行龐克是我搖滾魂的故鄉，出道15年後的今天，我想 LiSA 現在應該可以正面迎戰了。而且，田淵前輩（為 LiSA 提供了許多歌曲的 UNISON SQUARE GARDEN 田淵智也）過去所寄予的 LiSA 形象也是流行搖滾，所以我想在15週年重新正面去做流行龐克，要做這首只有我自己，就去做了。

——主唱自己創作的旋律，能把握住自己最順暢的音域走向，又能在上面適當地加點難度，這就是自作曲特有的魅力所在吧。LiSA 你如何評價作曲家身份的自己？

我一開始是因為憧憬 SPEED 而開始跳舞，然後遇見了 Avril Lavigne，透過搖滾越來越多喜歡的東西。再遇見動畫，LiSA 誕生了……把喜歡的人的喜歡之處全部集合裝飾起來，就成了 LiSA 吧。旋律我想也是如此，大家託付給 LiSA 的那些歌，我唱進自己的身體裡，才漸漸能唱出想唱的旋律。作為作曲家，我想我也是被大家培育出來的。

想讓日本妖怪起舞

——專輯中也實現了與新作曲家的合作。最令人驚訝的還是ケンモチヒデフミ提供的「小豆あらい feat. MAISONdes, ケンモチヒデフミ」。

是的（笑）。

——向作家性極強的ケンモチ委託作曲，前提就是要飛進「ケンモチ的世界」。在這個時間點這樣做，是因為什麼？

我一直覺得ケンモチ先生在某種意義上是絕對不會交集的地方。但去年在美國巡演時……在我憧憬的音樂之城美國，真切感受到當地的人們真的在享受日本的音樂。日式的享受方式，拿著螢光棒、喊著「オイ！オイ！」、大家一起做同樣的事，對美國人來說本來是不可能的事，但他們是作為日本文化在享受。從海外的視角看日本文化，就是和服、忍者，一直都沒什麼更新。既然我有很多機會能出國，如果能獲得在日本受歡迎的這個時機走向海外的角色，我就想讓全世界起舞。用日本的聲音讓他們起舞。想到能做到這件事的人，就是ケンモチ先生了。

——原來如此。是要用正確的方式向海外呈現異國風情。ケンモチ先生一定也嚇了一跳吧，「咦，要幫 LiSA 寫曲？」。

哈哈。ケンモチ先生的創意真的非常豐富，我只是給了他幾個關鍵字，第一次開會就說「小豆洗い怎麼樣？」。妖怪是日本文化，而小豆洗い在妖怪中又比較特殊，是個只會洗小豆、可愛的妖怪，他很認真地說這樣的妖怪也應該受到重視。我超愛紅豆餡，覺得根本就是量身打造（笑）。太棒了。

與 GLIM SPANKY 的合作，呈現全新的 LiSA

——GLIM SPANKY 提供的「SWEET MAGIC」在搖滾中帶有油漬搖滾的氣息，賦予了 LiSA 以往的搖滾所沒有的新寬度。為什麼在這個時間點與 GLIM SPANKY 合作？

過了10週年之後，我感覺自己已經超越了當初想像的 LiSA 形象，從那時候開始又在找讓自己興奮的音樂。我從 GLIM 很早期就一直在聽他們的音樂，但我一直覺得這樣的音樂由松尾レミ那樣的主唱來唱，絕對比我唱更帥。但超過10年後、想呈現全新的 LiSA 時，心態轉變成：讓喜歡的人幫自己寫喜歡的歌也無妨吧。跟キタニタツヤ、ツミキ這些讓我感動的音樂人商量的機會增加了，GLIM 就是其中之一。

——音樂上是喜歡的，但不覺得是自己要唱的？

對。我現在做的這種……怎麼說，狂放不羈的搖滾（笑），各種類型混合在一起的搖滾，我是透過動畫才認識的。在那之前更喜歡的是三和弦就夠的那種直球龐克，那個才是我的根本。但在那個領域得不到關注，所以我以為那不適合我，但當我能夠做自己喜歡的事的時候，覺得如果被允許的話現在就要去做了。

——專輯中段的「あばよ」和後段的「HELLO WORLD」，是 LiSA 作品中已是常客的小南泰葉的作曲。小南提供的歌曲，像是擔任了 LiSA 根源中帶有 Avril 味道的那個部分。

對。「あばよ」其實是相當久之前就寫好交給我的作品。《LACE UP》怎麼說呢，每首歌的味道都很濃烈。所以我想要有點像是可以拈起一顆來吃的豆子那樣的歌。一直是牛排和漢堡排，胃會受不了的，想來點和食。所以就說想把之前收到的「あばよ」放進去，但錄音錄著錄著就玩起來了（笑），結果變成了相當濃郁的那種和食。

——聽起來音符不多、是簡單的搖滾，但段落的轉換意外地多。

對對。「HELLO WORLD」是在發完上一張《LANDER》（2022年11月發行的第6張專輯）之後……疫情出口逐漸看得見的時候，期望著能對新世界說聲「你好」時誕生的歌。

——說到新作家的組合，「HOMEだよ」也令人驚訝。沒想到有一天いきものがかり和 LiSA 會站在同一個舞台上。

我也想說「沒想到能走到這一步」。水野（良樹）先生的いきものがかり的歌，我喜歡「気まぐれロマンティック」和「笑顔」，那種貼近日常的溫暖氛圍是我寫不出來的，所以我想借助水野先生的力量，做一首能坦率傳達心意的歌，就去找他商量了。

——把專輯中抒情曲的位置完全交給水野先生。這種 J-POP 正統抒情曲，由迎接15週年的現在的 LiSA 來唱，我覺得非常好。

我也這麼覺得。嗯。因為年歲增長，因為走過了15年……當初像我一樣想從家裡逃出來而來看演唱會的孩子們，長大成人了，出社會了，有些成了爸爸媽媽，帶著孩子一起來看我的演唱會，也有人因為《鬼滅之刃》而與曾經關係不好的家人一起來。走過了15年才看得見的大家的故事，我也希望有一首能讓那樣的大家唱給家人聽的歌，這就是「HOMEだよ」的誕生背景。

——作詞是水野先生和 LiSA 共同創作，但「HOMEだよ」這個標題應該是水野先生取的吧。感覺從 LiSA 這邊不太會出現……

對呢（笑）。

——「残酷な夜に輝け」「ブラックボックス」「ReawakeR（feat. Felix of Stray Kids）」這三首味道極其濃烈的已發行歌曲之後，第12首「シャドウ」是繼「ハルシネイト」（2024年5月發行的單曲《Shouted Serenade》收錄曲）之後ツミキ的第二首提供曲。ツミキ最近也以 Aooo 的鼓手身份備受矚目，身為全能音樂人、作曲編曲家同樣鋒芒畢露，各方面都讓人感受到一種天才氣質。

真的。我以為他是鼓手，所以應該擅長搖滾樂隊風格的音樂，但饒舌的段落又很有舞曲的感覺。我問他為什麼能做這樣的東西，他說其實也非常喜歡黑人音樂。原來還有那條線，真的什麼都能做，太驚訝了。順帶一提，這首歌的貝斯由 Aooo 的やまもとひかる彈奏，她的 slap bass 怎麼說呢，不是那種陽剛的感覺，有種像是從遊戲裡走出來的女孩的可愛，真的很棒。

在現場遇見的衝擊，才是最棒的音樂體驗

——LiSA 你至今仍定期以 CD 這個實體形式製作並發行專輯。前陣子ナタリー刊登了 DREAMS COME TRUE 中村正人的專訪，很有意思。受邀的嘉賓映秀。生於2002年，對「專輯是什麼」這件事真的完全不懂。LiSA 你對專輯和實體的堅持是什麼？

串流普及之後，包括我自己，也變得更容易被發現了。就算是在英國只在小場地演出、完全沒有知名度的藝術家，用串流也能找到很多符合喜好的音樂。那也成為我的養分，同樣地，別人也可能因此找到我。完全不瞭解動畫的人也能找到 LiSA，對這件事我感受到很深的希望。但我也算是昭和的人（笑），喜歡的藝術家的 CD 我還是會買。因為想要那個實體。

——DREAMS COME TRUE 這次特意先行限定 CD 發行，讓串流往後，說是希望優先讓大家有「聆聽完整一張專輯」的體驗，想讓人透過 CD 來聽。

田淵前輩的個人翻唱專輯也只出 CD。透過串流認識的音樂，不像一邊看歌詞本一邊聽的歌那樣能跟著唱。以前那種一邊讀歌詞一邊「我也想像這個人一樣唱」的事，現在沒辦法做了，所以這點確實需要思考。但我覺得，自己在接觸新音樂時感受到最大衝擊的體驗，是現場演出。就算 CD 賣不起來，在現場遇見的歌曲帶來衝擊，連同記憶留在那個人心中，成為屬於那個人的歌，這讓我最開心。等到哪天帶著興趣找到了專輯，那時候再去享受其中的故事吧。帶著好奇心去享受，對於專輯而言，那樣才是最讓我開心的事。

搞不好一下子就到18週年了

——攜帶《LACE UP》的15週年巡演將於4月的武道館揭幕……這個「LiVE is Smile Always～15～（iCHiNiTSUiTE GO）」的標題，第一眼看到時「這在寫什麼？」但其實就是「各就定位，預備，出發！」吧。

對（笑）。畢竟是15週年的起跑。

——就像在壓軸「Patch Walk」中唱的「歩き続けて また会いましょう」一樣，關於15週年，比起「終於走到了這裡」，更大的心情是「要從這裡出發」？

對。走過了14週年，準備好出發了。巡演作為15週年的起跑……雖然是專輯的巡演，但首先還是想好好地跟大家一起慶祝15年。我要辦一個好的演出。「PATCH WALK」的延續吧，希望能讓遇見我的大家想起遇見時的那個時光。

——目前公告的有國內巡演、6月起的亞洲巡演，以及歐洲巡演。我想其中一定有許多還無法公開的計畫……

今年這一年，在嘗試許多從來沒做過的事、同時實現這15年間浮現的那些好點子的過程中……

……搞不好一下子就到18週年了（笑）。

——已經有相當充實的計畫了。

對。開跑之後，大家應該也會一眨眼發現過了3年，所以請好好跟上來（笑）。

——有沒有什麼最近帶給你新刺激、可能通往未來表現的東西？

讓我想想……心跳率吧。

——心跳率？

聽說2026年是適合開始新事物的一年，不只是我，是地球上的所有人。我也想著要開始些新的事，考慮過爬山或騎摩托車之類的，但受傷了會讓很多計畫崩盤。感冒也不行，開始什麼事卻影響到身體的話就本末倒置了。想著在注意的同時讓自己更強大，需要提升的是體力，能不能讓這件事也變得有趣，然後找到的就是熱量訓練。提高心跳率，把它維持在平常演唱會後半段左右的水準，看能撐多久。

——哦。那種追求記錄的苦行式鍛鍊，本來就是你喜歡的嗎？

不是。我是個超沒耐性的人，沒辦法把時間花在跟 LiSA 無關的事情上。熱量訓練是能直接讓 LiSA 變得更好的東西，所以才去做。

——對於無法為 LiSA 的活動帶來正向幫助的事情，大部分就直接捨棄了？

對。無法替 LiSA 加分的事，大概都不太會去做吧。
