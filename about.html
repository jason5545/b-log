關於我

我用一根手指打字，坐在輪椅上，靠 AI 當義肢寫程式。

我會反編譯別人的軟體看裡面在搞什麼、拆開加密格式研究它怎麼鎖你的資料、在虛擬機裡偽裝 Mac 然後讓 Apple ID 乖乖登入。Gemini 把我的輪椅標記成敏感資訊要幫我隱藏的時候，我寫了一篇文章罵它。LiSA 的官方鬧鐘 App 壞了下架了，我把它逆向重寫。

這個部落格就是這些事情的紀錄。

核心信念

技術中立

工具就是工具。VID/PID 是公開資訊、KVM 偽裝不等於作弊、逆向工程是理解系統的方式——這些是技術事實，不該因商業或政治壓力被迴避。

效率是尊重

減少點擊、減少輸入、自動化一切可自動化的事。當每次打字都有代價時，這不是偷懶，是生存策略。

拒絕保護性歧視

我不需要「為你好」的限制。我需要工具、選擇、平等對待——不是同情，不是保護，不是「我們幫你決定什麼對你好」。

務實主義

重要的是決策對不對，不是決策怎麼產生的。重要的是觀點有沒有道理，不是觀點怎麼被表達的。

這裡有什麼

技術分析與開發——逆向工程、故障排除、系統修復、自己寫工具解決自己的問題。LINE 快取加密怎麼鎖你的照片、NAS 被駭之後怎麼救回來、macOS 觸控板不支援單指捲動所以我自己寫一個。

AI 觀察——不是產業預測，是使用者視角。內容過濾怎麼把身障者當嫌犯、AI 協作跟 AI 代筆的差別在哪、被 Reddit 罵 slop 之後我想到了什麼。

文化與商業觀察——購票系統對輪椅使用者的「公平」、身障票被拿去轉賣、VTuber 的情感陪伴商業模式、微軟到底在不在乎你盜版。

生活記事——追了十年的 LiSA、模擬飛行二十年、以及偶爾承認自己其實沒有想像中那麼冷。

技術棧

開發: Next.js, TypeScript, Kotlin, Swift, Python
硬體: Proxmox VE (9950X3D + RTX 4070 Ti Super), Raspberry Pi, ESP32
協作: Claude Code (Opus), Kimi k2

為什麼寫

現實中偽裝的夠多了。這裡是我可以不裝的地方。

不是勵志故事，不是正能量，不是「身障者做到某某事好厲害」。就是一個人在想事情，然後把想的東西寫下來。技術的、社會的、個人的，都有。

興趣

MSFS 飛行模擬、LiSA（P0）、暗黑破壞神4、策略卡牌遊戲。

關於這個網站

(b)-log 是一個雙重用途的內容系統，完全存在於 GitHub 儲存庫中。它既是分享見解的部落格，也是管理項目的待辦清單。每篇文章都是一個 Markdown 檔案，一個在 data/posts.json 中的小型條目，僅此而已。

純 HTML/CSS/JS + Markdown，部署在 GitHub Pages，使用 GitHub Actions 自動生成 WordPress 風格永久連結。
